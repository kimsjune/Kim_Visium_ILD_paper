# Kim_Visium_ILD_paper
This repo contains code for [paper]().

"figs" directory contains Jupyter notebooks or R markdown documents (rendered to html) that directly produce figure panels. Files are organized based on which figure (`f` for main figure, `sf` for supplementary figure) they appear in the manuscript. Some files are duplicated if they produce panels appearing in a main and supplementary figure. Figure-panel reference (ex. Fig. 1b)  is written at the top of each document in markdown.  

"misc" dicrectory contains all other code indirectly processing files required to run code in files in "figs".
